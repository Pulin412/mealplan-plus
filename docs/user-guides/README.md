# MealPlan+ User Guide

This folder contains platform-specific user guides for MealPlan+.

## Available guides

- [Android User Guide](./android-user-guide.md)
- [iOS User Guide](./ios-user-guide.md)
- [Web User Guide (Coming Soon)](./web-user-guide.md)

## Which guide should I use?

- Use the Android guide if you run MealPlan+ on Android phones/tablets.
- Use the iOS guide if you run MealPlan+ on iPhone/iPad.
- Use the Web guide for release updates and expected browser experience.

## Core feature areas covered

- Account setup and sign-in
- Home dashboard overview
- Meal planning and calendar
- Daily logging
- Diet templates
- Health metrics
- Grocery lists
- Profile/settings and data actions
