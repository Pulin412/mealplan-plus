# MealPlan+ iOS User Guide

## 1. Before You Start

- Install and open the iOS app.
- Create an account from **Create Account** or log in.
- Returning users may open directly to main tabs.

## 2. Sign In and Account Access

### Sign in

1. Open MealPlan+.
2. Enter email/password.
3. Tap **Sign In**.

### Create account

1. On login, tap **Sign Up**.
2. Fill required fields.
3. Tap **Create Account**.

### Forgot password

1. Tap **Forgot Password?**.
2. Enter email and submit.

Note: Password reset email behavior depends on app auth mode; if local-only, password recovery may be limited.

## 3. Bottom Navigation

iOS tab bar:

- **Home**
- **Meal Plan**
- **Log**
- **Diets**
- **Health**
- **Grocery**

## 4. Home Dashboard

Use **Home** to view:

- Daily calorie target progress
- Macro cards/rings
- This week activity
- Today’s plan slots
- Summary stats (A1C/weight/streak)

From Home, you can open profile and plan/log actions.

## 5. Meal Plan

Use **Meal Plan** to assign diets on calendar dates.

### Main actions

1. Tap date in month/week view.
2. Tap **Select Diet** or **Change**.
3. Review selected-date panel (macros, slots, tags).
4. Open **View Log** for the selected date.

### Views

- Toggle between **Month** and **Week**.
- Date cells show planned/completed/today states.

## 6. Food Log

Use **Log** (Food Log) to record daily intake.

### Main actions

1. Move between dates.
2. Assign or clear diet for selected day.
3. Add foods by slot.
4. Complete or reopen day when needed.
5. Compare **Daily Log** and **Plan vs Actual** tabs.

## 7. Diets

Use **Diets** to create and reuse diet templates.

### Main actions

1. Tap **+** in My Diets.
2. Build a diet with meal slots and meals.
3. Save and assign in Meal Plan/Log.
4. Edit, duplicate, or delete existing diets.

### Filtering

- Search by diet name.
- Filter by tags.
- Optional ingredient-based filter.

## 8. Health Metrics

Use **Health** to track measurements.

### Main actions

1. Tap **+** to add metric.
2. Select metric type (for example weight, blood sugar).
3. Save values and review summaries/charts.
4. Open **History** for details by metric type.

## 9. Grocery Lists

Use **Grocery** to manage shopping lists.

### Main actions

1. Tap **+** to create a list.
2. Open a list to view items and progress.
3. Delete lists with swipe/delete actions.

## 10. Settings and Profile

### Settings

Settings include:

- Daily calorie goal
- Dark mode toggle
- Meal reminder toggle
- Data actions like sample data import

### Profile

Profile allows updating:

- Personal and body metrics
- Lifestyle and goals
- Nutrition target calories

Actions include:

- Save profile
- Logout
- Clear all data (diets/grocery/health)
- Delete account flow

## 11. Troubleshooting

- If data appears missing, pull to refresh on list screens.
- If tabs do not jump to expected date, reopen Log tab and retry.
- If import is running, wait for completion message before leaving screen.
