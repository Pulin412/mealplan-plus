# MealPlan+ Web User Guide

## Status

The MealPlan+ web application is **coming soon**.

## Planned Scope

The first web release is expected to include:

- Account sign-in/sign-up
- Home dashboard
- Meal planning calendar
- Daily log
- Diet templates
- Health metrics
- Grocery list management

## Current Recommendation

Until web is available, use:

- Android app for full feature access.
- iOS app for current mobile feature set.

## What to Expect Next

When web is released, this guide will be updated with:

- Browser requirements
- First-time setup
- Web navigation
- Feature-by-feature walkthrough
- Known limitations and FAQ
