# MealPlan+ Android User Guide

## 1. Before You Start

- Install and open the Android app.
- Create an account from **Sign Up** or log in with an existing account.
- If you already signed in before, the app may open directly on **Home**.

## 2. Sign In and Account Access

### Sign in

1. Open the app.
2. Enter your email and password.
3. Tap **Sign In**.

### Create account

1. Tap **Sign Up**.
2. Fill name, email, password, and confirm password.
3. Tap **Create Account**.

### Forgot password

1. Tap **Forgot Password?** on login.
2. Enter your email.
3. Tap **Send Password**.

Note: Passwords are stored locally and cannot be emailed from the app.

## 3. Bottom Navigation

Android bottom tabs:

- **Home**
- **Meal Plan**
- **Log**
- **Diets**
- **Health**
- **Grocery**

## 4. Home Dashboard

Use **Home** to quickly check:

- Today calorie progress
- Macro rings (carbs/protein/fat/calories)
- This week activity
- Today’s plan slot progress
- Key stats (e.g., A1C, weight, streak)

From Home, you can jump to daily logging and planned meal details.

## 5. Meal Plan (Calendar)

Use **Meal Plan** to assign diets to dates and inspect planned meals.

### Main actions

1. Tap a date in the calendar.
2. Tap **Select Diet** / **Change** / **+ Plan** depending on date status.
3. Review daily macro totals and meal slots.
4. Use **View Log** to open that date in logging.

### Views

- Switch between **Month** and **Week**.
- Calendar cells indicate planned/completed/today/no-plan states.

## 6. Daily Log

Use **Log** to track what you eat each day.

### Main actions

1. Select date (left/right arrows).
2. Tap diet icon to assign a diet for the selected date.
3. Expand meal slots and tap **Add Food**.
4. Log quantity and unit.
5. Optional: clear plan, complete day, or reopen day.

### Useful views

- **Daily Log** tab: slot-by-slot items.
- **Plan vs Actual** tab: compare planned vs consumed macros.

## 7. Diets

Use **Diets** to manage reusable full-day plans.

### Main actions

1. Tap **New Diet**.
2. Set name/description and tags.
3. Assign meals to slots.
4. Save and reuse in Meal Plan or Daily Log.

### Filtering

- Search by name.
- Filter by tags.
- Use advanced filters for ingredient and slot.

## 8. Health Metrics

Use **Health** to track readings and trends.

### Main actions

1. Tap **Log Reading**.
2. Select metric type.
3. Save value/date.
4. Review trend chart and period stats.

Features include period filters (7/14/30 days), stats cards, trend chart, and recent readings list.

## 9. Grocery Lists

Use **Grocery** to organize shopping.

### Main actions

1. Tap **+** to create a list.
2. Open a list to manage items.
3. Mark items complete.
4. Delete lists from the overflow menu.

## 10. Profile and Settings

### Profile

Open profile from Home avatar.

You can update:

- Personal details (name, age, gender)
- Body metrics (weight, height)
- Lifestyle and goal
- Daily calorie target

You can also:

- **Save Profile**
- **Logout**
- Use danger-zone actions (e.g., clear all data)

### Settings

Settings include:

- Theme controls (system/dark mode/dynamic color on supported Android)
- Data export (food log, health metrics)
- Data import (JSON/CSV diets)

## 11. Troubleshooting

- If a screen looks stale, switch tabs and return or relaunch the app.
- If login fails, verify email/password first.
- If import fails, confirm file format and columns.
- For export/share issues, check Android share permissions and available apps.
