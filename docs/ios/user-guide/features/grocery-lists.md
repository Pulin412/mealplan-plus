# Grocery Lists — iOS

Generate shopping lists from your meal plan or create them manually.

---

## Auto-generate from Plan

1. Go to **More → Grocery → +**.
2. Choose **Generate from Plan** and select a date range.
3. The app aggregates all foods needed across your planned diets for that period.
4. Tap **Generate**.

---

## Manual List

1. Tap **+** → **Manual**.
2. Enter a list name and add items one by one.

---

## Managing Items

- Tap an item to check it off as you shop.
- Items are grouped by category (Vegetables, Grains, Proteins, etc.).
- The progress bar shows how many items are checked.
