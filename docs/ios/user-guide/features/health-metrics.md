# Health Metrics — iOS

Track weight, blood glucose, blood pressure, HbA1c, or any custom measurement over time.

---

## Logging a Measurement

1. Go to the **Health** tab → tap **+**.
2. Select the metric type.
3. Enter the value (and sub-type for blood glucose: Fasting / Post-meal / Random).
4. Set date and time → tap **Save**.

---

## Custom Metric Types

Go to **Health → Manage Types → +** to define your own metric with a name, unit, and optional normal range.

---

## Charts

Tap **View Charts** to see line charts of any metric over time. Select the metric type and date range using the controls at the top.

---

## Blood Glucose Sub-types

| Sub-type | When to measure |
|----------|----------------|
| Fasting | Morning before eating |
| Post-meal | 1-2 hours after a meal |
| Random | Any other time |
